Exercice 3 :

Écrire une fonction qui prend en entrée un entier n et retourne une liste de tous les nombres premiers inférieurs ou égaux à n.

Solution Ex3 :
def nombres_premiers(n):
   # Initialisation de la liste des nombres premiers
   premiers = []
    # Parcours de tous les nombres de 2 à n
   for i in range(2, n+1):
       # Vérification si i est un nombre premier
       est_premier = True
    for j in range(2, int(i**0.5)+1):
       if i % j == 0:
          est_premier = False
          break
       # Si i est premier, on l'ajoute à la liste
       if est_premier:
          premiers.append(i)
    return premiers

Exercice 4 :

Écrire une fonction qui prend en entrée une liste de nombres et retourne la médiane de la liste.

Solution Ex4 :
def mediane(liste):
    # Tri de la liste
    liste_triee = sorted(liste)
    n = len(liste_triee)
   # Cas où la liste a un nombre pair d'éléments
   if n % 2 == 0:
       m = n // 2
       # Calcul de la moyenne des deux éléments du milieu
       med = (liste_triee[m-1] + liste_triee[m]) / 2
    # Cas où la liste a un nombre impair d'éléments
   else:
       m = (n+1) // 2
       # L'élément du milieu est la médiane
       med = liste_triee[m-1]
    return med

Exercice 5 :

Écrire une fonction qui prend en entrée une chaîne de caractères et retourne True si la chaîne est un palindrome (c'est-à-dire qu'elle peut être lue de la même façon de gauche à droite et de droite à gauche), et False sinon.

Solution Ex5 :
def est_palindrome(chaine):
    # Conversion de la chaîne en minuscules et suppression des espaces
    chaine = chaine.lower().replace(" ", "")
    # Comparaison de la chaîne avec sa version inversée
   return chaine == chaine[::-1]

Exercice 6 :

Écrire une fonction qui prend en entrée une liste de nombres et retourne le nombre de valeurs qui sont plus grandes que la moyenne de la liste.

Solution Ex6 :
def nb_valeurs_sup_moyenne(liste):
   # Calcul de la moyenne de la liste
   moyenne = sum(liste) / len(liste)
   # Comptage des valeurs supérieures à la moyenne
   nb_valeurs_sup = len([x for x in liste if x > moyenne])
   return nb_valeurs_sup

Exercice 7 :

Écrire une fonction qui prend en entrée deux listes de nombres et retourne une nouvelle liste qui contient les nombres qui sont présents dans les deux listes.

Solution Ex7 :
def intersection_listes(liste1, liste2):
    # Utilisation de la fonction set.intersection() pour trouver les éléments communs
    intersection = set(liste1).intersection(set(liste2))
    # Conversion de l'intersection en liste et tri par ordre croissant
   intersection_liste = sorted(list(intersection))
   return intersection_liste

Exercice 8 :

Écrire une fonction qui prend en entrée une chaîne de caractères et retourne le nombre de fois que chaque caractère apparaît dans la chaîne.

Solution Ex8 :
def compter_caracteres(chaine):
   # Initialisation d'un dictionnaire vide
   comptes = {}
   # Parcours de chaque caractère de la chaîne
   for char in chaine:
   # Si le caractère est déjà dans le dictionnaire, on incrémente son compteur
    if char in comptes:
          comptes[char] += 1
       # Sinon, on ajoute une entrée pour le caractère avec un compteur initialisé à 1
      else:
          comptes[char] = 1
    return comptes

Exercice 9 :

Écrire une fonction qui prend en entrée une liste de nombres et retourne la liste de toutes les paires de nombres dont la somme est égale à un nombre donné.

Solution Ex9 :
def trouver_paires(liste, cible):
    # Initialisation d'un ensemble vide pour stocker les paires trouvées
    paires = set()
    # Parcours de chaque nombre dans la liste
   for i, x in enumerate(liste):
       # Parcours de chaque nombre restant dans la liste
       for j, y in enumerate(liste[i+1:], start=i+1):
          # Si la somme des deux nombres est égale à la cible, on ajoute la paire à l'ensemble
          if x + y == cible:
             paires.add((x, y))
   # Conversion de l'ensemble en liste et retour
   return list(paires)
