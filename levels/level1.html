Exercice 1 :

Écrire un programme qui calcule la somme de deux nombres donnés par l'utilisateur.

Solution Ex1 :
a = float(input("Entrez le premier nombre: "))
b = float(input("Entrez le deuxième nombre: "))
somme = a + b
print("La somme de", a, "et", b, "est égale à", somme)

Exercice 2 :

Écrire un programme qui calcule la moyenne de trois nombres donnés par l'utilisateur.

Solution Ex2 :
a = float(input("Entrez le premier nombre: "))
b = float(input("Entrez le deuxième nombre: "))
c = float(input("Entrez le troisième nombre: "))
moyenne = (a + b + c) / 3
print("La moyenne de", a, ",", b, "et", c, "est égale à", moyenne)

Exercice 3 :

Écrire un programme qui convertit une température de Fahrenheit en Celsius.

Solution Ex3 :
fahrenheit = float(input("Entrez la température en degrés Fahrenheit: "))
celsius = (fahrenheit - 32) * 5 / 9
print("La température en degrés Celsius est de", celsius)

Exercice 4 :

Écrire un programme qui calcule la factorielle d'un nombre donné par l'utilisateur.

Solution Ex4 :
n = int(input("Entrez un entier positif: "))
fact = 1
for i in range(1, n+1):
fact *= i
print("La factorielle de", n, "est", fact)

Exercice 6 :

Écrire un programme qui trouve le plus grand nombre parmi trois nombres donnés par l'utilisateur.

Solution Ex6 :
a = float(input("Entrez le premier nombre: "))
b = float(input("Entrez le deuxième nombre: "))
c = float(input("Entrez le troisième nombre: "))
if a > b and a > c:
print(a, "est le plus grand nombre")
elif b > a and b > c:
print(b, "est le plus grand nombre")
else:
print(c, "est le plus grand nombre")

Exercice 7 :

Écrire un programme qui calcule la somme des chiffres d'un nombre entier donné par l'utilisateur.

Solution Ex7 :
n = int(input("Entrez un nombre entier: "))
somme = 0
while n != 0:
somme += n % 10
n //= 10
print("La somme des chiffres du nombre est", somme)

Exercice 8 :

Écrire un programme qui affiche les nombres premiers entre 2 et un nombre donné par l'utilisateur.

Solution Ex8 :
n = int(input("Entrez un nombre entier: "))
print("Les nombres premiers entre 2 et", n, "sont:")
for i in range(2, n+1):
for j in range(2, i):
if i % j == 0:
break
else:
print(i)
