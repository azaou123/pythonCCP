Exercice 1 :

Écrire une fonction qui prend en entrée une liste d'entiers et renvoie la somme de tous les éléments de la liste.

Solution Ex1 :
def sum_list(lst):
   return sum(lst)

Exercice 2 :

Écrire une fonction qui prend en entrée une liste d'entiers et renvoie la moyenne de tous les éléments de la liste.

Solution Ex2 :
def mean_list(lst):
   return sum(lst) / len(lst)

Exercice 3 :

Écrire une fonction qui prend en entrée une liste d'entiers et renvoie le plus grand élément de la liste.

Solution Ex3 :
def max_list(lst):
   return max(lst)

Exercice 4 :

Écrire une fonction qui prend en entrée une liste d'entiers et renvoie le plus petit élément de la liste.

Solution Ex4 :
def min_list(lst):
   return min(lst)

Exercice 5 :

Écrire une fonction qui prend en entrée une liste d'entiers et renvoie une nouvelle liste avec tous les éléments multipliés par 2.

Solution Ex5 :
def double_list(lst):
   return [x * 2 for x in lst]

Exercice 6 :

Écrire une fonction qui prend en entrée deux listes d'entiers et renvoie une nouvelle liste avec tous les éléments communs aux deux listes.

Solution Ex6 :
def common_elements(lst1, lst2):
   return list(set(lst1) & set(lst2))

Exercice 7 :

Écrire une fonction qui prend en entrée une liste d'entiers et renvoie une nouvelle liste avec tous les éléments pairs de la liste.

Solution Ex7 :
def even_elements(lst):
   return [x for x in lst if x % 2 == 0]

Exercice 8 :

Écrire une fonction qui prend en entrée une liste d'entiers et renvoie une nouvelle liste avec tous les éléments impairs de la liste.

Solution Ex8 :
def odd_elements(lst):
   return [x for x in lst if x % 2 != 0]

Exercice 9 :

Écrire une fonction qui prend en entrée une liste d'entiers et renvoie une nouvelle liste avec tous les éléments triés par ordre croissant.

Solution Ex9 :
def sort_list(lst):
   return sorted(lst)

Exercice 10 :

Écrire une fonction qui prend en entrée une liste d'entiers et renvoie une nouvelle liste avec tous les éléments triés par ordre décroissant.

Solution Ex 10 :
def sort_list_desc(lst):
   return sorted(lst, reverse=True)
